# Lupus Disease Prediction using Machine Learning

## Overview

This project focuses on predicting Lupus disease using Machine Learning techniques and healthcare datasets. The system analyzes patient health parameters and predicts the likelihood of Lupus disease occurrence.

The project was developed as part of a multidisciplinary team project and demonstrates the application of Artificial Intelligence in healthcare.

---

## Problem Statement

Lupus is a complex autoimmune disease that can be difficult to diagnose early.

The objective of this project is to utilize Machine Learning techniques to assist in identifying disease patterns and improve prediction accuracy.

## Machine Learning Model

The project uses the Random Forest algorithm for classification and disease prediction.

### Why Random Forest?

- High accuracy
- Handles large datasets
- Reduces overfitting
- Suitable for healthcare prediction systems
